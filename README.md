# Hyperbridge Token

The Hyperbridge token project consists of 3 areas:

* Smart Contracts (Ethereum) - in the `smart-contracts` dir
* Communication API (JSON.API) - in the `communication-api` dir
* Documentation - in the `documentation` dir
